# praiseme
r package to deliver praise to the user

<h1>installation</h1>
install using devtools
```
library(devtools)
devtools::install_github("perikarya/praiseme")
```

<h1>examples</h1>

```library(praiseme)
praiseme()
#> "You're great!"
```

```library(praiseme)
praiseme("art")
#> "You're great at art!"
```
